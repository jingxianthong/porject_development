Install Screen Reader Extension

To enhance accessibility and enable screen reading capabilities, you can install a screen reader extension for your browser.

We recommend installing the Screen Reader extension from the Chrome Web Store.

Once installed, the extension will provide screen reading functionality, allowing you to navigate and access content on websites more easily.

Enable Window Narrator

Press the Windows logo key + Ctrl + Enter to turn on Narrator. This is the default shortcut to enable Narrator.
